# Workstation Needs Reality Check

Status:

```text
Scope recalibration note
```

Date:

```text
2026-07-03
```

## Question

Are the current workstation candidates too high-end for the user's actual
project scope?

## Short Answer

```text
Yes, the search has drifted higher than the actual scope requires.
```

The current project scope is:

- ChatGPT
- Codex
- VS Code
- GitHub
- Markdown repositories
- Microsoft Office
- repository management
- documentation
- research
- software development
- information processing

This is a primary productivity and development workstation need. It is not
currently a local AI training workstation, gaming PC, video production rig,
large virtual machine host, or multi-GPU compute box.

## What The Actual Need Supports

The actual need supports a machine that is:

- fast and stable for everyday development
- comfortable with many browser tabs, Office files, VS Code, Git, and Codex
- reliable enough to be the primary work machine
- wireless-competent
- easy to maintain
- expected to feel good for 3-5 years

It does not require chasing every newest AI-branded CPU or every configuration
above the normal hard target.

## Recalibrated Specification

### Practical Target

This should be the default search lane:

- completed cost around $1,000
- acceptable stretch to $1,300 when the value is obvious
- Windows 11 Pro
- 32 GB RAM acceptable at purchase if upgradeable
- 64 GB RAM preferred, not mandatory on day one
- 1 TB NVMe acceptable if expansion is easy
- 2 TB NVMe preferred when price difference is reasonable
- Ryzen 7 / Ryzen 9 / Intel Core Ultra 7 / Intel Core Ultra 9 all acceptable
- Wi-Fi 6E acceptable; Wi-Fi 7 preferred
- direct or trusted retailer support path

### Normal Ceiling

The normal hard target remains:

```text
$1,500 completed cost
```

But candidates near $1,500 should be treated as premium normal-band choices,
not the new default.

### Exceptional Lane

The exceptional visibility band remains:

```text
$1,500-$2,000
```

But it should be used only when the candidate provides a specific, concrete
benefit the practical lane does not provide. Examples:

- substantially better warranty or return safety
- no-upgrade 64 GB / 2 TB configuration
- much stronger thermals or reliability evidence
- materially better port/wireless capability needed by the workflow
- unusually strong sale price relative to typical pricing

It should not be used just because a better processor exists.

## What Was Overweighted

The search began overweighting:

- newest AI CPU branding
- 96 GB RAM
- 2 TB as a hard day-one requirement
- exceptional-band candidates
- performance headroom beyond the stated workflow

These are useful signals, but they should not dominate the decision.

## What Should Be Reweighted

Increase weight on:

- completed cost
- ready-to-use stability
- seller / warranty / return path
- quiet, reliable everyday operation
- enough RAM and storage, not maximum RAM and storage
- direct usefulness to Codex, VS Code, repositories, Office, and browser-heavy work

Decrease weight on:

- newest CPU generation for its own sake
- AI TOPS as a headline metric
- 96 GB memory unless there is a known local workload
- above-$1,500 candidates unless they clearly beat cheaper options

## Recalibrated Candidate Interpretation

### Best Fit Zone

Candidates in the $800-$1,300 range with Windows 11 Pro, 32 GB RAM, 1-2 TB SSD,
strong CPU, and a clean RAM/storage upgrade path are likely the realistic fit.

Examples from current evidence:

- GEEKOM GT1 Mega AI 32 GB / 2 TB, if the about-$1,299 price validates
- GEEKOM IT15 AI 32 GB / 1 TB, if the lower price validates and storage upgrade
  is easy
- GEEKOM A8 Max AI 32 GB / 1 TB, if value and upgrade path validate
- GEEKOM IT13 Max 16 GB / 1 TB only if RAM upgrade cost makes the completed
  system clearly cheaper than 32 GB alternatives

### Premium Normal Zone

Candidates around $1,400-$1,500 are acceptable only when they are clearly
cleaner or more complete than the $1,000-$1,300 options.

Examples:

- GEEKOM IT15 AI 32 GB / 2 TB at $1,499
- GEEKOM A9 Max HX 370 32 GB / 2 TB at $1,499

### Exceptional Zone

Candidates above $1,500 should not lead unless the user explicitly chooses to
pay for extra certainty or extra headroom.

Examples:

- MINISFORUM AI X1 Pro-370 96 GB / 2 TB
- GEEKOM A9 Max HX 470 if price remains above $1,500

## Recalibrated Procurement Posture

Decision posture:

```text
Wait, but narrow the search downward.
```

Recommended search correction:

```text
Focus on $800-$1,300 practical candidates first.
Treat $1,300-$1,500 as premium normal-band candidates.
Treat $1,500-$2,000 as exceptions only.
```

The current system does not need to buy the most powerful machine visible. It
needs to buy the least expensive machine that will remain comfortable, reliable,
and low-friction for the stated work.
