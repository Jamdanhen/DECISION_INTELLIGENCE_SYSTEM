## Device Connectivity Baseline

The GEEKOM IT15 can support a multi-monitor setup through:

- 2 HDMI ports
- 2 USB4 ports with display output support
- Wi-Fi 7
- Bluetooth 5.4
- 2.5 GbE Ethernet
- multiple USB-A ports
- SD card reader
- 3.5 mm audio

Practical display rule:

```text
Use HDMI for simple 4K/60 monitors. Use USB-C/USB4 to DisplayPort or USB-C
monitor input when higher-quality display handling or cleaner cabling matters.
```

## Recommended Starting Setup

Best practical starting point:

- One good 27-inch or 32-inch 4K monitor.
- Wired Ethernet if the desk location allows it.
- Existing keyboard and mouse if available.
- One external backup drive or SSD.
- Surge protector or small UPS.

Do not buy a USB hub immediately unless the actual desk setup needs more ports.

## Monitor Options

### Best single-monitor setup

```text
27-inch or 32-inch 4K IPS monitor, 60 Hz or better
```

Why:

- Best fit for coding, documents, browser research, ChatGPT/Codex, and Office.
- 4K gives more usable workspace than 1080p.
- A 27-inch 4K monitor is very sharp.
- A 32-inch 4K monitor gives more comfortable scaling and larger text.

Connection:

- HDMI 2.0 or HDMI 2.1 cable to HDMI port; or
- USB-C/USB4 to DisplayPort cable; or
- USB-C cable to a USB-C monitor that supports DisplayPort Alt Mode.

### Best dual-monitor setup

```text
Two 27-inch monitors, either 1440p or 4K
```

Recommended layout:

- Main monitor: 27-inch or 32-inch 4K.
- Secondary monitor: existing monitor, 27-inch 1440p, or portable monitor.

Connection:

- Monitor 1 through HDMI.
- Monitor 2 through HDMI or USB-C/USB4 to DisplayPort.

### Portable or travel option

```text
15-16 inch USB-C portable monitor
```

Use when:

- The workstation may move between locations.
- A second screen is useful but a permanent second desk monitor is not needed.

## USB Hub / Dock Need

Not required on day one.

The IT15 already has enough ports for a normal setup:

- keyboard/mouse receiver or wired keyboard
- webcam
- external backup drive
- monitor connections
- Ethernet

Buy a hub only if one of these becomes true:

- more than 3-4 USB devices need to stay plugged in
- the monitor has no useful USB ports
- a single desk cable/docking setup is desired
- an external SSD, webcam, microphone, keyboard, mouse, and card reader all need
  to be connected at once

If a hub is needed, prefer:

```text
Powered USB-C hub or USB4 dock with at least 100W power pass-through, USB-A,
USB-C, Ethernet, and DisplayPort/HDMI.
```

Avoid:

- very cheap unpowered hubs for permanent desktop use
- hubs that claim too many display features without clear 4K/60 support
- unknown-brand docks if they will become the center of the workstation

## Other Peripherals

Recommended:

- Ethernet cable, Cat 6 or better.
- Surge protector or compact UPS.
- External SSD or external hard drive for local backup.
- Webcam if the monitor does not include one.
- Speakers or headset if needed.
- VESA mount only if mounting the mini PC behind the monitor helps desk layout.

Optional:

- KVM switch if sharing one keyboard, mouse, and monitor with the existing
  laptop.
- USB-C portable monitor for second-location work.
- Monitor arm for better ergonomics.

## First Purchase Priority

Recommended order:

1. Monitor.
2. Ethernet cable and surge protection or UPS.
3. Backup drive.
4. Keyboard/mouse only if existing ones are not comfortable.
5. USB hub or dock only after the actual desk cable problem is known.

## Do Not Overbuy

Do not start with a large dock, expensive high-refresh gaming monitor, or
specialized multi-monitor hardware unless the real desk layout requires it.

The IT15's integrated graphics and ports are excellent for productivity
multi-monitor work, but this setup should prioritize readable workspace,
stability, and backup/recovery over gaming-style display specs.
