## 3. Current Market Summary

The practical market is good enough. The issue is no longer finding power; it is
avoiding unnecessary spend.

Key conclusions:

- 32 GB RAM is the right practical floor.
- 64 GB RAM should remain a preference, not a day-one requirement.
- 16 GB systems should not lead unless completed upgrade cost is verified.
- 1 TB storage is acceptable when expansion is easy.
- 2 TB is worth paying for when the added cost is modest, especially because the
  machine is intended to act as a work hub.
- Current storage usage on Sleeper suggests 1 TB is enough for now if GitHub and
  backups are handled correctly.
- Above-$1,500 candidates should be exceptions only.
- Network stability, backup/sync behavior, storage reliability, and recovery
  path now matter more than raw processor class.

Current DDR5 SODIMM pricing is high enough that immediate RAM-upgrade plans are
risky. This makes 32 GB included more important than a cheap 16 GB base price.
